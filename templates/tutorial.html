Beginners Flask Tutorial

So I'm going to be showing you guys how to create a basic Flask app to get you started, it's only going to be the setup for the Python code so no HTML, CSS etc will be provided. To get that you're going to have to do the dirty work of reading the docs.

Installing Flask

The first thing we want to do is go into the terminal and instal Flask. To do this we are going to use pip. Remember, if you're on windows use pip, and if you're on mac use pip3. This should take a few seconds and it'll prompt you it's done. 

# Mac
pip3 install flask
#Windows
pip install flask
Importing flask

Now that we've installed Flask, all we have to do now is import it to include it in our app. Remember, the reason we import Flask from flask is because Flask is an object of the flask library. 

from flask import Flask
Creating an instance of Flask

The next thing we want to do is create an instance of flask. The first and only argument is the name of our app's module or package. Since we are only using one file, we are going to put __name__ as our argument. More on this here. It's important to understand what exactly __name__ is and what I mean by us only using one file, so I recommend you watch the video as we are going to be using __name__ again later. 

from flask import Flask
     
app = Flask(__name__) #Just added
Creating the route

Now, we are finally set up to begin creating the page. We are first going to use the route() director. This basically gives our page a URL to reference from other pages, and this URL will trigger the function. We then create a function and give it a name. In this case I am creating a homepage so I'm going to give it the name home. It's best practice to just give your function the same name as the route above it. Now that we've created a function, we are going to need to return something. In Flask what we return is what we see on the page. This could be a string, or an html file. In our case we don't want to overcomplicate things, so we will return a string saying "I am home". This string will be what we see on the page. 

from flask import Flask
     
app = Flask(__name__)
    
@app.route("/home") #This app.route() and function just added
def home(): 
    return "I am home"
Running the app

Your basic webpage is finished! All that's left to do is run it. We first want to include a conditional checking to see if we are on our 'main file' so to speak. Since we only have one file this is true, so we will then run the app with app.run(). The argument taken in basically allows us to edit the app and refresh the page without having to run the script all over again. Now just go to your command line, run the program, and voila! Your first Flask website! I will be leaving links to the docs and some useful resources for learning Flask below so you can get started on learning how to REALLY make flask programs. Best of luck to you all! 

from flask import Flask
     
app = Flask(__name__)
    
@app.route("/home") #This app.route() and function just added
def home(): 
    return "I am home"
    
if __name__ == '__main__':
    app.run(debug=True)
python3 site.py #Assuming we have named the app site.py, but just insert your filename there instead.
