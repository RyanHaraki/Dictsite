About the Site

EasyDict was created entirely in Python using the Flask and Difflib framework. The CSS framework Bootstrap was used for the frontend. I decided to use Bootstrap because I find personally find the frontend to be rather tedious, and Bootstrap provides a clean, overall good looking design without having to put in too much extra effort. The site took approximately 10-12 hours to make.

About the Dev

I'm a 17 year old currently living in Vancouver, BC. Since we are currently quarantined because of COVID-19 I decided I wanted to really work on my coding abilities, so I decided to make my first real useful application without the use of a tutorial. This resulted in EasyDict, where I took some old code, learned how to use Flask, and turned my code into a web application.
